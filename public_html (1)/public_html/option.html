Tag HTML <option>

Exemplo

Uma lista suspensa com quatro opções:

                   
  

   <label for="cars">Choose a car: </label>
 
   <select id="cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
        <option value="opel">Opel</option>
        <option value="audi">Audi</option>
   </select>

Definição e Uso

A <option> (Word Break Opportunity) Define uma opção em uma lista de seleção.

Elementos <option> vão dentro de um elemento <select>,<optgroup>,<datalist>.

Observação: a tag <option> pode ser usada sem quaisquer atributos, mas geralmente você precisa do atributo value, que indica o que é enviado ao servidor no envio do formulario.

Dica: se você tiver uma longa lista de opções, poderá agrupar opções relacionadas na tag <optgroup>.

Atributos

Atributo Valor Descrição
disabled disabled Especifica quando uma opção deve ser desabilidata
label text Especifica um pequeno rótulo para uma opção
selected selected Especifica qual opção deve está pré-selecionada quando a página carregar
value text Especifica o valor que vai ser enviado ao servidor
