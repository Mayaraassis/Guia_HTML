Tag HTML <strong>

Exemplo

A tag <strong> define um texto importante em um documento:

                  
<strong>Isso é um texto importante!</strong>

Definição e Uso

A tag <strong> é usada para definir o texto com grande importância. O conteúdo interno é normalmente exibido em negrito.

Dica: Use a tag <b> para especificar texto em negrito sem qualquer importância extra.

Configurações Padrão de CSS

A maioria dos navegadores exibirá o <strong> elemento com os seguintes valores padrão:


strong {
    font-weight: bold;
}
