Tag HTML <style>

Exemplo

O uso de <style> aplica uma folha de estilo simples a um documento HTML:

    
<html>
<head>
<style>
    h1 {color:red;}
    p {color:blue;}
</style>
</head>
<body>

<h1>Um heading.</h1>
<p>Um parágrafo.</p>

</body>
</html>
</code>

Definição e Uso

A tag <style> é usado para definir uma informação de estilo (CSS) para um documento.

Dentro da tag <style>, você específica como os elementos HTML devem ser processados em um navegador.

Dicas e Notas

Nota: Quando um navegador lê uma folha de estilo, ele formata o documento HTML de acordo com as informações na folha de estilo. Se algumas propriedades foram definidas para o mesmo seletor (elemento) em diferentes folhas de estilo, o valor da última folha de estilo lida será usado (veja o exemplo abaixo).

Dica: para vincular a uma folha de estilo externa, use a tag <link>.

Atributos

Atributo Valor Descrição
media media_query Especifica em qual dispositivo o documento vinculado vai ser exibido.
type text/css Especifica o tipo de mídia da tag <style>.

Mais Exemplos

Exemplo

Vários estilos para o mesmo elemento.

                    
<html>
<head>
<style>
    h1 {color:red;}
    p {color:blue;}
</style>
<style>
    h1 {color:gren;}
    p {color:pink;}
</style>
</head>
<body>

<h1>Isso é um heading</h1>
<p>Isso é um parágrafo.</p>

</body>
</html>
</code>

Configurações Padrão de CSS

A maioria dos navegadores exibirá o <style> elemento com os seguintes valores padrão:

                    
style {
    display: none;
}
