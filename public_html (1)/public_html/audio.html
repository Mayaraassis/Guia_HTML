Tag HTML <audio>

Exemplo

Reproduz um arquivo de som:

                   
<audio controls>
    <source src="horse.ogg" type="audio/ogg">
    <source src="horse.mp3" type="audio/mpeg">
</audio>

Definição e Uso

A tag <audio> é usada para incorporar conteúdo de som em um documento, como música ou outros fluxos de áudio.

A tag <audio> contém uma ou mais <source>tags com diferentes fontes de áudio. O navegador escolherá a primeira fonte que suporta.

O texto entre as tags <audio> e </audio>só será exibido em navegadores que não suportam a tag <audio>.

Existem três formatos de áudio suportados em HTML: MP3, WAV e OGG.

Dicas e Notas

Dica: Para arquivos de vídeo, use a tag <video>.

Atributos

Atributo Valor Descrição
autoplay autoplay Especifica que o aúdio vai reproduzir assim que ele estiver pronto.
controls controls Especifica que os controles de aúdio devem ser mostrado (tais como o botão de play/pause).
loop loop Especifica que o aúdio vai começar a tocar novamente, toda vez em que ele terminar.
muted muted Especifica que a saída de áudio deve ser silenciada.
preload auto
metadata
none		 Especifica se e como o autor acha que o áudio deve ser carregado quando a página carrega.
src URL Especifica a URL do arquivo de aúdio.
