Tag HTML <s>

Exemplo

A tag <s> marca um texto que não está mais correto

                  
<p><s>Apenas 2 ingressos restantes!</s></p>
<p>VENDIDO!</p>

Definição e Uso

A tag <s> é usada para definir o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha atravessada.

A <s>tag não deve ser usada para definir o texto excluído em um documento, use a tag <del> para isso.

Configurações Padrão de CSS

A maioria dos navegadores exibirá o <s> elemento com os seguintes valores padrão:


s {                    
  text-decoration: line-through;
}
