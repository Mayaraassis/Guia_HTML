Tag HTML <link>

Exemplo

Faz uma ligação com uma folha de estilos(CSS).


<head>
<link rel="stylesheet" href="styles.css">
</head>

Definição e Uso

A tag <link> define uma ligação entre o documento atual e uma fonte externa.

A tag <link> é frequentemente usada para fazer uma ligação entre folhas de estilos externas

A tag <link> é um elemento vazio e contém apenas atributos.
