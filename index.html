Objetivo do Website

Este é um guia que deve ser usado como referência por qualquer um que deseje aprender HiperText Markup Language ou Linguagem de Marcação de Hipertexto. É através do HTML que conseguimos inserir conteúdo e definir a estrutura de um site. O HTML não é uma linguagem de programação, isso significa que não pode ser usado para criar funcionalidades dinâmicas. Entretanto, o HTML possibilita a organização e formatação de documentos, similar ao Microsoft Word. Os navegadores têm a capacidade de ler e renderizar um arquivo com a extensão .html possibilitando assim que o usuário final consiga visualizar a página criada. Este Guia não abrange todas as tags HTML, você pode ver a lista de tags clicando no botão abaixo.
