Tag HTML <source>

Exemplo

Um reprodutor de áudio com dois arquivos de origem. O navegar vai escolher o primeiro <source> compatível.

                   
<audio controls>

    <source src="horse.ogg" type="audio/ogg">
    <source src="horse.mp3" type="audio/mpeg">
   
</audio>

Definição e Uso

A tag <source> é usada para especificar multiplos recursos de mídia para elementos como <video>, <audio> e <picture>.

A tag <source> permite que você especifique arquivos alternativos de vídeo/áudio/imagem dos quais são escolhidos de acordo com o suporte do navegador ou de acordo com a largura da janela de visualização. O navegador escolherá a primeira <source> compátivel.

Atributos

Atributo Valor Descrição
media media_query Aceita qualquer media query válida que normalmente seria definida em um CSS.
sizes Especifica tamanhos de imagem para diferentes layouts de página.
src URL Obrigatório quando <source> é usado em <audio> e <video>. Especifica o URL do arquivo de mídia.
srcset URL >Obrigatório quando <source> é usado em <picture>. Especifica o URL da imagem para usar em diferentes situações.
type MIME-type Especifica o tipo MIME do recurso.
