Tags HTML

Tag Descrição

<link>

Define uma relação entre um documento e uma fonte externa (mais usado para vincular folhas de estilos).

<style>

Define informações de estilo para um documento

<strong>

Define um texto importante em um documento.

<dialog>

Define uma caixa ou janela de diálogo.

<audio>

Define uma informação de som embutida.

<s>

Define um texto que não está mais correto.

<time>

Define uma hora específica.

<body>

Define o corpo do documento.

<source>

Define vários recursos de mídia para elementos de video e audio

<caption>

Define a legenda da tabela.

<thead>

Agrupa o conteúdo do cabeçalho em uma tabela.

<template>

Define um contêiner para o conteúdo que deve ser escondido quando a página carrega.

<wbr>

Define um ponto de quebra de linha em um texto caso o parágrafo seja muito longo.

<option>

Define uma opção em uma lista de seleção.

<code>

Define um pedaço de código de computador.

<i>

Exibe conteúdo interno em itálico

<div>

Define uma divisão ou seção em um documento HTML.

<sub>

Define um textosubscrito.
