Tag HTML <caption>

Exemplo

Uma tabela com legenda:

                   


<table>

    <caption> Economia mensal</caption>


    <tr>
        <th>Mes</th>
        <th>Economia</th>
    </tr>

    <tr>
        <td>Janeiro</td>
        <td>$100</td>
    </tr>


</table>

Definição e Uso

A tag <caption> define a legenda da tabela.

A tag <caption> deve ser inserida imediatamente após a tag <table>.

Configurações CSS predefinidas

A maioria dos navegadores, por padrão, exibirá o elemento <caption> com os seguintes valores: 

                   
caption {

    display: table-caption;
    text-align:center;

    }
