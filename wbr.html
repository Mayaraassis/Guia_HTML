Tag HTML <wbr>

Exemplo

Um texto com prováveis quebras de linhas:

                   
  <p>To learn AJAX, you must be familiar with the XML<wbr>Http<wbr> Request Object.</p>

Definição e Uso

A <wbr> (Word Break Opportunity) específica onde em um texto seria correto adicionar uma quebra de linha.

Dica: quando a palavra é muito longa, o navegador pode quebrá-la no lugar erroado. Você pode usar o <wbr> para adicionar oportunidades de quebras de linha.
