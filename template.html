Tag HTML <template>

Exemplo

Use <template> para manter algum conteúdo que ficará oculto quando a página carregar. Use JavaScript para exibi-lo: 

                   
<button onclick="showContent()"> Mostrar conteudo escondido</button>

<template>

    <h2>Flores</h2>
    <img src="img_white_flower.jpg" width="214" height="204">

<script>

function showContent(){

    var tem = document.getElementByTagName("template")[0];
    var clon = temp.content.cloneNode(true);
    document.body.appendChield(clon);
    
}
</script>

Definição e Uso

A tag <template> é usada como um contêiner que mantém algum conteúdo HTML oculto do usuário quando a página é carregada;

O conteúdo dentro da tag <template> pode ser renderizado posteriormente com JavaScript.

Você pode usar a tag <template> se tiver algum código HTML que deseja usar repetidamente, mas não antes de solicitá-lo. Para fazer isso sem a tag <template>, você deve criar o código HTML com JavaScript para evitar que o navegador renderize o código.
