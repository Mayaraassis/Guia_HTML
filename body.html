Tag HTML <body>

Exemplo

Um simples documento HTML:

                   
<html>

<head>

    <title> Titulo do documento</title>

</head>

<body>

    <h1> Este é um cabeçalho</h1>
    <p> Este é um cabeçalho</p>

</body>

</html>

Definição e Uso

A tag <body> Define o corpo do documento.

A tag <body> contém todo o conteúdo de um documento HTML, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

Configurações CSS predefinidas

A maioria dos navegadores, por padrão, exibirá o elemento <body> com os seguintes valores: 

                   
body {

    display: block;
    margin:8px;

    }
  
body:focus {
    
     outline:none;
}
