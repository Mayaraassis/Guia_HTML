Tag HTML <thead>

Exemplo

Uma tabela HTML com os elementos <thead>, <tbody> e <tfoot>:

                   


<table>

    <thead>
        <tr>
            <th>Mes</th>
            <th>Economia</th>
        </tr>
    </thead>
  
    <tbody>
        <tr>
            <td>Janeiro</td>
            <td>$100</td>
        </tr>

        <tr>
        <td>Fevereiro</td>
        <td>$80</td>
    </tr>
    </tbody>

    <tfoot>
        <tr>
            <td>Soma</td>
            <td>$180</td>
        </tr>
    </tfoot>

</table>

Definição e Uso

A tag <thead> é usada para agrupar p conteúdo do cabeçalho de uma tabela HTML.

A tag <caption> é usada em conjunto com com as tags <tbody> e <tfoot> para especificar cada parte da tabela (cabeçalho, corpo e rodapé).

Os navegadores podem usar estes elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e rodapé. Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.

Nota: O elemento <thead> deve conter uma ou mais tags <tr>

A tag <thead> deve ser usada no seguinte contexto: Como filho de um elemento <table>, após qualquer elemento <caption> e <colgrup> e antes de qualquer elemento <tbody>, <tfoot> e <tr>.

Configurações CSS predefinidas

A maioria dos navegadores, por padrão, exibirá o elemento <thead> com os seguintes valores: 

                   
thead {

    display: table-header-group;
    vertical-align:center;
    border-color: inherit;

    }
