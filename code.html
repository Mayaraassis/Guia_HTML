Exemplo

Defina algum texto como código de computador em um documento:

                     

<p>The HTML <code>button</code> tag defines a clickable button.</p>
<p>
  The CSS <code>background-color</code> 
  property defines the background color of an element.

Definição e Uso

A tag <code> tag é usada para definir um pedaço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Dica: esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS.
