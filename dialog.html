Tag HTML <dialog>

Exemplo

Usando a tag <dialog>:

                    
<dialog open>Esta é uma janela de diálogo aberta.</dialog>

Definição e Uso

A tag <dialog> define uma caixa de diálogo ou subjanela.

A tag <dialog> facilita a criação de diálogos pop-up e modais em uma página web.

Atributos

Atributo Valor Descrição
open open Especifica que o elemento <dialog> está ativo e que o usuário pode interagir com ele.
